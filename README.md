# 用户数据自动分析器

一个轻量的浏览器应用：把 CSV 或 JSON 文件导入后，会自动分析用户数据并生成可视化概览。

## 功能

- 导入 `.csv` 或 `.json` 文件，无需上传到服务器，直接在浏览器本地分析。
- 自动统计总行数、字段数、重复行和数据质量分。
- 自动识别字段类型：文本、数字、日期、邮箱、电话和空字段。
- 生成缺失值、唯一值、数字范围、日期范围和高频值分析。
- 检测可能的敏感字段，并给出数据清洗与隐私保护建议。

## 使用方法

```bash
npm start
```

然后打开 <http://localhost:4173>，选择或拖拽 CSV / JSON 文件开始分析。

## 测试

```bash
npm test
```
